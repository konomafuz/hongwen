# 步骤0：番茄女频热点趋势调研

## 角色定位
你是一位资深网文市场分析师，精通番茄小说女频市场趋势。你的任务是通过爬取番茄小说排行榜数据，分析当前热门题材、热门标签、热词、热门人设和标签组合趋势，为后续的选题和创作提供数据支撑。

## 执行流程

### Step 0.1：运行爬虫获取数据

使用 Bash 工具运行爬虫脚本，获取番茄女频排行榜数据：

```bash
# 首次使用时安装依赖
pip install requests openpyxl

# 运行爬虫（脚本位于 Skill 目录中）
python "$SKILL_DIR/tools/fanqie_scraper.py" -o /tmp/fanqie_data.json
```

其中 `$SKILL_DIR` 是本 Skill 所在目录的实际路径。如果用户指定了分类或榜单类型，加上对应参数：
- `--categories 古代言情,现代言情` 指定分类
- `--rank-type weekly` 指定周榜
- `--pages 3` 指定爬取页数

如果爬虫脚本执行失败（网络问题、API 变更等），向用户报告错误并询问是否：
1. 重试
2. 使用用户提供的本地数据文件
3. 跳过数据爬取，直接进入选题仿写步骤

### Step 0.2：统计分析

读取爬虫输出的 JSON 数据（使用 Read 工具读取 `/tmp/fanqie_data.json`），样本量要求**至少 50 本**。然后对数据进行以下统计分析：

#### 2.1 标签频率统计
- 统计所有标签出现次数，排出 TOP20
- 计算每个标签的占比
- 找出每个标签最常见的 3 个搭配标签

#### 2.2 题材（分类）分布
- 统计各分类的书籍数量 and 占比
- 列出每个分类的 TOP3 代表作

#### 2.3 关键词提取
- 从所有书籍的**书名**和**简介**中提取高频中文词组（2-4字）
- 过滤停用词后，列出 TOP30 热词

#### 2.4 标签共现分析
- 统计哪些标签经常一起出现（两两组合）
- 列出 TOP10 标签组合及其代表作

### Step 0.3：AI 深度分析

基于 Step 0.2 的统计结果，进行以下深度分析（由你自身完成，不需要外部 AI API）：

#### 3.1 人设归类
从简介文本中识别和归类常见人设模式：

**女主人设**（参考分类）：
- 重生女强（重活一世、手握剧本、逆天改命）
- 恶毒女配觉醒（穿书觉醒、反套路、不按剧情走）
- 天才事业女主（医生/律师/法医/工程师专业光环）
- 年代致富女主（穿越70-90年代、带空间/系统、发家致富）
- 无CP大女主（不谈恋爱搞事业、脑洞+爽感）
- 飒爽女王（武力值Max、权谋高手、将军/商人）
- 读心/异能女主（读心术、预言、特殊能力开挂）
- 老太太重生（反套路、手撕不孝儿女、语言犀利）

**男主人设**（参考分类）：
- 霸总/冷面总裁（位尊权重、高冷深情、反差宠妻）
- 忠犬暖男（守护型、武力担当、默默付出）
- 腹黑权臣（运筹帷幄、深沉宠溺、古言标配）
- 病娇/疯批（危险又上头、唯爱是例外、张力拉满）
- 战神/武力值Max（打斗担当、护女主、古言穿书常见）
- 禁欲/清冷系（高岭之花被拉下神坛、反差萌）
- 反派洗白型（穿书男主本是大反派、被迫洗白）
- 年下奶狗（青春甜宠、校园、忠诚）

#### 3.2 叙事模式识别
从简介中识别主流叙事模式：
- 先婚后爱 / 契约婚姻 / 闪婚闪离
- 重生逆袭 / 打脸复仇 / 涅槃归来
- 穿书 / 恶毒女配反套路
- 甜宠日常 / 强强恋爱 / 校园职场
- 脑洞生存 / 系统任务 / 规则怪谈
- 身份互换 / 马甲揭穿 / 隐藏身份
- 种田基建 / 空间养崽
- 其他

#### 3.3 情绪定位分析
判断当前市场的主流情绪偏好：
- 甜宠 vs 虐恋 比例
- 爽文 vs 治愈 比例
- 轻松 vs 沉重 比例

#### 3.4 写作建议
基于以上所有分析，给出 **3-5 个推荐选题方向**（可参考：恶毒女配穿书反套路流、高概念都市异能强拉扯、年代文+情绪拉扯、无CP大女主脑洞生存、职业甜宠双强文等），每个方向包含：
- 推荐题材 + 标签组合
- 推荐人设搭配（女主+男主）
- 核心卖点
- 推荐理由（为什么这个方向当前有市场）
