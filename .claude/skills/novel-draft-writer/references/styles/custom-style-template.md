# 临时模仿文风提炼模板

当用户粘贴文章、提供片段或要求“模仿这段文风”时，先按本模板提炼临时文风，再用于正文创作。

## 提炼维度

1. 叙事视角：贴近谁、信息给多少、是否强内心OS。
2. 句式节奏：长短句比例、停顿方式、是否常用省略/反问/倒装。
3. 语气色调：冷、甜、疯、克制、毒舌、沙雕、雅致等。
4. 情绪推进：靠动作、对白、心理、环境，还是反差推进。
5. 细节颗粒：偏生活化、诗性、冷硬、密集吐槽或强爽点反馈。
6. 禁忌：哪些表达不能照搬，哪些风格特征不适合本书设定。

## 输出格式

```markdown
【临时模仿文风】
- 文风名称：{可选名称}
- 核心气质：{一句话概括}
- 句式节奏：{具体说明}
- 语气色调：{具体说明}
- 情绪推进：{具体说明}
- 可迁移写法：{3-5条}
- 禁止照搬：{需要避开的原文痕迹或不适配元素}
```
