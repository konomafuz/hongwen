# 🪶 红文织梦 (Hongwen Zhimeng)

![红文织梦](docs/banner.png)

> **“世事洞明皆学问，人情练达即文章。”**
>
> 每一个题材都是一个待醒之梦，只待你落笔生花。

**红文织梦** 是一款为网文作家、短剧编剧量身打造的**沉浸式终端创作助手**。无需繁琐的技术配置，它能帮你从选题推荐、大纲构筑，一路陪伴至正文撰写、自动润色与 Word 规范排版，让灵感在终端中自由流淌。

---

## 🌟 为什么选择红文织梦？（四大创作优势）

*   **🎨 懂你的多体裁模板**
    无论是情感拉扯的女频长篇、爽点密布的男频连载，还是高概念反转的悬疑短篇、节奏拉满的竖屏短剧，它都能懂你的设定，生成最符合品类期待的文章。
*   **💡 拒绝“AI腔”，行文有颗粒感**
    内置“写作者 -> 编辑质检 -> 精修重写”的智能纠偏循环，砍掉注水与翻译腔，自动匹配网文热梗与爆款金句小抄，写出接地气、有画面感的好故事。
*   **⚡️ 极速上手，一键排版**
    章节创作完成后，自动按规范合成为 Word 文档。特别是竖屏短剧，会自动排版为“场景居中、台词加粗、动作置灰”的专业剧本格式，直接交付制片方。
*   **🔒 守护隐私，安全无忧**
    您的 API Key 仅保存在本地 `.env` 文件中，并已被 Git 自动忽略，绝不会上传或泄露给任何人。

## 🚀 极速起航（快速下载与使用）

### 1. 下载项目
在终端中克隆仓库并进入文件夹：
```bash
git clone https://github.com/konomafuz/hongwen.git
cd hongwen
```

### 2. 一键启动
为了让您快速开始，我们已经为您配置好了极简的启动口令。在项目目录下直接输入：

```bash
hongwen
```
*(Windows Powershell 下如提示权限限制，可输入 `.\hongwen` 启动；您也可以通过 `python novel.py` 运行。)*

## 🧭 创作工坊指令指南

在红文织梦的交互界面中，只需输入以下简单的汉字/拼音指令，即可玩转创作：

| 创作指令 | 魔法效果 🪄 |
| :--- | :--- |
| **`init`** | **初始化新作品**。挑选您喜欢的体裁。如果您没有灵感，在书名处输入 `r`，AI 会为您精心推荐 3 个爆款选题。首次使用时还会引导您安全导入 API 密钥。 |
| **`write [章数]`** | **执笔创作章节**（如 `write 1`）。系统会自动规划这一章的大纲、备好词汇小抄、撰写初稿并由 AI 编辑精修，一气呵成。 |
| **`export`** | **一键导出大作**。自动读取书名，将已创作的章节拼装成高品质的 Word `.docx` 电子书或剧本。 |
| **`status`** | **查看大作状态**。快速纵览书名、体裁、已写章节进度与总字数。 |
| **`style`** | **切换行文风格**（如“现言甜宠”、“古言雅致”等）。 |
| **`trend fetch`** | **联网抓取今日爆款数据**，帮您分析当下的创作趋势与流量密码。 |
| **`help`** | 呼出您的创作指南秘籍。 |
| **`exit`** | 封笔歇墨，保存进度并安全退出。 |
